# 💳 Credit Card Fraud Detection

## 📌 Description
This project detects fraudulent credit card transactions using Machine Learning techniques.

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## 🚀 Steps
1. Load dataset
2. Preprocess data
3. Handle class imbalance
4. Train model (Logistic Regression)
5. Predict transactions
6. Evaluate using precision, recall, F1-score

## 📊 Output
The model identifies fraudulent transactions with high recall and precision.
